## 项目和小破站

- **[阿猹的小窝](https://acha.xx.kg)**  
  我的个人网站，拿来放博客、项目、音乐碎碎念和各种奇奇怪怪的小玩意儿。

- **[Groceries Box](https://acha.xx.kg/groceries/)**  
  我的网站杂货箱。里面放了原创小工具、小网页游戏，还有一些我自己部署或者收录整理过的东西。

- **[ASCII Art Converter](https://acha.xx.kg/ascii/)**  
  一个图片转字符画的小工具，核心是 `Canvas` 像素提取。后来又给它补了底色切换、一键复制和移动端适配。

- **[Emoji Combiner](https://acha.xx.kg/emoji/)**  
  基于 `Cloudflare Workers AI` 和 `Vercel` 的脑洞小玩具，会把不同类别的 Emoji 搓成一个新物种，还顺便胡编一段设定。

- **[ACha_'s Post Office | 猹の邮局](https://mail.acha.xx.kg)**  
  自建邮箱系统。基于 `Cloudflare Email Routing`、`Workers`、`D1` 和 `R2` 搭的，算是把“想要一个自己域名邮箱”这件事真做出来了。

- **[Emoji Minesweeper](https://acha.xx.kg/minesweeper/)**  
  表情包扫雷。看起来很抽象，玩起来更抽象，很适合拿来摸鱼。

- **[Hot Air Balloon](https://acha.xx.kg/balloon/)**  
  一个复刻 Google Play 断网彩蛋风格的小程序。轻量，但挺容易玩上头。

- **[Tinyhttpd-Refactored](https://github.com/ACha-xwx/Tinyhttpd-Refactored)**  
  和课程相关的 C 语言 Web 服务器重构实践，也是把我重新拽回“认真维护博客”状态的项目之一。

## 我除了写代码，还真的会写歌

- 从 `2020.03.15` 开始认真接触音乐制作，最早是从 `Undertale` Remix 和同人曲一路摸进来的。
- 主要写 `EDM`，尤其喜欢 `Uplifting Trance`、`Progressive House`、`House`、`Melodic Bass`、`Liquid Drum 'n Bass` 和 `Euphoric Hardstyle`。
- 有时候也会研究管弦编排，偶尔尝试写一点日文歌词。
- 代表作和方向包括 `Daybreak`、`Drown in Abyss`，以及那首一口气混了四种风格的新年曲 `Dream Voyager`。
